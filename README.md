# 🌍 Wanderlust - Travel Booking App

## 🚀 Overview
Wanderlust is a full-stack AI-powered travel booking web application that allows users to explore destinations, plan trips, and manage bookings efficiently.

## ✨ Features
- 🌐 Explore travel destinations
- 📅 Plan trips with start & end dates
- 💰 Entry fee & transport cost calculation
- 🧾 Booking management system
- 🤖 AI-based travel recommendations (coming soon)

## 🛠 Tech Stack
- Frontend: React.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- AI Integration: (Planned)

## 📂 Project Structure
- `/frontend` → React application
- `/backend` → API and server
